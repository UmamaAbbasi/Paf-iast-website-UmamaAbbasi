![PAF-IAST Logo](https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcRhqoVsh0DC5ISve4XcuS6lqKen6mPe-MZFRw&s)

## 📋 Project Overview
A modern, responsive one-page website for **Pak-Austria Fachhochschule: Institute of Applied Sciences and Technology (PAF-IAST). Created for Web Engineering course assignment.

## ✨ Features
- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ CSS animations (fade-in, floating logo, pulse effects)
- ✅ Interactive program filters (All, Computing, Engineering, Life Sciences)
- ✅ Working contact form with validation
- ✅ Dynamic breadcrumb navigation
- ✅ Smooth scrolling between sections
- ✅ Mobile-friendly hamburger menu
- ✅ Faculty profiles with social links
- ✅ LMS portal integration
- ✅ Separate contact information section

## 🛠️ Technologies
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- Font Awesome 6
- Google Fonts (Poppins)
- FormSubmit.co for email handling
